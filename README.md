# eCommerce Frontend Design

## Project Overview
This project is a realistic and professional frontend implementation of an eCommerce website, inspired by a provided Figma design. It focuses on clean UI, modern layouts, and good UX, built entirely for desktop resolution. 

## Technologies Used
- HTML5 (Semantic Structure)
- CSS3 (Vanilla CSS with Flexbox/Grid, CSS Variables)
- Vanilla JavaScript (Interactivity)

## Weekly Implementation Breakdown

### Week 1 - Structure & Layout
- **Header design**: Implemented a responsive navigation bar with a logo, functional-looking search bar, and user action icons.
- **Footer design**: Created a multi-column footer with links to various sections (About, Partnership, Information, etc.) and social media icons.
- **Base layout setup**: Configured global CSS variables, typography (Inter font), standardized container width, and reset styles.

### Week 2 - Core Pages
- **Home page implementation (`index.html`)**: Features a modern hero section with a banner and CTA, a featured categories grid, and a promotional product carousel/grid showcasing items with hover effects.
- **Product listing page (`products.html`)**: Implemented a robust CSS Grid layout for products, alongside a sidebar containing mock filter options (category, brand, price). Added product cards displaying images, titles, pricing, and ratings.

### Week 3 - Product Details & Interactivity
- **Product detail page (`product-detail.html`)**: Features a product image gallery (main image and thumbnails), detailed product information (price, options, rating), and description tabs.
- **JavaScript interactivity features (`main.js`)**: Added simple interactions including customized dropdowns for product options (Size/Color), search bar mock behavior, and image gallery swapping.
